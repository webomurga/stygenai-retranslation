## Experimental Corpus Organization

This study uses two primary experimental corpus partitions to support comparative stylometric analysis of human translations and machine translation systems.

To ensure reproducibility of the experiments reported in the paper, the dataset is organized into two independent corpus roots:

```text
corpus_data.human/
corpus_data.nmt/
```

Each corpus root contains its own aligned `_txt` and `_morph` directories, keyword files, generated vectors, and analysis outputs.

---

## 1. `corpus_data.human/`

This dataset contains:

* original human literary translations,
* reference corpora,
* and outputs generated by NMT systems trained on human translator data.

Example structure:

```text
corpus_data.human/
├── ae_txt/
├── ae_morph/
├── bd_txt/
├── bd_morph/
├── kk_txt/
├── kk_morph/
├── machine_txt/
├── machine_morph/
├── ref_txt/
└── ref_morph/
```

This configuration is primarily used for:

* Human vs Human stylistic comparison,
* Human vs Human-trained NMT analysis,
* measuring whether an NMT system reproduces translator-specific stylistic traits.

---

## 2. `corpus_data.nmt/`

This dataset contains corpora associated with machine-generated translation systems.

Example structure:

```text
corpus_data.nmt/
├── human_txt/
├── human_morph/
├── machine_txt/
├── machine_morph/
├── llm_txt/
├── llm_morph/
├── ref_txt/
└── ref_morph/
```

These datasets are used for:

* Human vs Commercial-NMT analysis,
* Human vs LLM-NMT analysis,
* NMT vs NMT comparison,
* cross-model stylistic attribution experiments.

---

## Experimental Context

As part of the study, three separate OPUS-NMT systems were trained using:

* human literary translations,
* commercial NMT translations,
* LLM-generated translations.

The resulting systems were then compared against:

* the human translators whose texts were used during training,
* other NMT systems,
* and reference corpora.

This structure allows the same stylometric and authorship attribution pipeline to be reused across both traditional literary translation analysis and neural machine translation style evaluation experiments.
